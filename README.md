# MyLinuxSetup

## installation & usage

```sh
git clone https://github.com/wasabi-neko/MylinuxSetup
```

- install zsh with zim and powerlevel10k:
  - `cd ./zsh && zsh ./install-p10k.zsh`

- install zsh with oh-my-zsh and some useful packages
  - `cd ./ && ./autoSetUp.sh`
